Smartdown Stress-testing via Impress (EXPERIMENTAL)

This example incorporates many of the Smartdown Gallery examples and splats them together into a single Impress.js presentation. It's hard on the browser, but when it works it is pretty cool. It is intended reveal issues where Smartdown is not operating properly in certain contexts. The Gallery cards have not been adjusted to fit in the Slides, so there will be some vertical scrolling necessary.
